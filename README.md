# QueueSync

QueueSync is a MERN-stack clinic queue management system designed for clinics and polyclinics. It helps manage patient queues, appointments, billing, doctor availability, and real-time wait-time tracking through a web-based platform.

## Features

* Real-time patient queue management
* Doctor-wise queue tracking
* Appointment scheduling
* Receptionist dashboard
* Billing and payment management
* Patient wait-time estimation
* Role-based authentication
* Clinic administration dashboard

## Tech Stack

### Frontend

* React
* Vite
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm start
```
